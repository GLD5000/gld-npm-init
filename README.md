# npm-init
A custom init CLI App to update your package.json and README.md ready for publising on NPM
## Updating your package.json
- name
- version
- description
- main
- repository
- keywords
- author
- license
- type
- bugs
- homepage
- bin
## Updating your README.md
- Installation
- Example Usage
  - Import
  - Run
  - Example Input
  - Example Output  
- Update
- Uninstall
- Execute Directly from NPM
- License
